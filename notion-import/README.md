# AgroConnect Notion Import Files

These files are prepared for manual import into Notion.

## Files

- `AgroConnect-Project-Hub.md` - Main project hub page.
- `AgroConnect-Tasks.csv` - Task/backlog database.
- `AgroConnect-X-Content.csv` - Build-in-public content tracker.
- `AgroConnect-Milestones.csv` - Milestone tracker.
- `AgroConnect-Kanban-Board.csv` - Simple three-column Backlog, In Progress, and Done table.

## Suggested Notion Setup

1. Create a new page in Notion named `AgroConnect Project Hub`.
2. Import `AgroConnect-Project-Hub.md` into that page, or copy its content into the page.
3. Import `AgroConnect-Tasks.csv` as a database.
4. Import `AgroConnect-X-Content.csv` as a database.
5. Import `AgroConnect-Milestones.csv` as a database.
6. Optional: import `AgroConnect-Kanban-Board.csv` if you prefer a simple table with separate columns for Backlog, In Progress, and Done.

## Recommended Views

For `AgroConnect-Tasks.csv`:

- Board view grouped by `Status`.
- Table view sorted by `Priority`.
- Filtered view for `Status` is `Next` or `In Progress`.

For `AgroConnect-X-Content.csv`:

- Board view grouped by `Status`.
- Table view grouped by `Feature Area`.

For `AgroConnect-Milestones.csv`:

- Timeline or table view sorted by `Date`.
- Filtered view for upcoming milestones where `Status` is `Next`.
